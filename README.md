# 📊 Repositorio de Macroeconomía Interactiva

> [!IMPORTANT]
> **Project Spec-Anchored:** Este proyecto se rige por [GEMINI.md](./GEMINI.md). Cualquier modificación debe respetar la fuente de verdad (CORE Economics) y los estándares Apple-style.

## 📑 Descripción General
Plataforma educativa de alta fidelidad para el Grado en Economía. Contiene simuladores dinámicos que visualizan modelos macroeconómicos de corto y medio plazo.

**URL Pública:** [https://joseantoniobouortells.github.io/intro-macroeconomia/](https://joseantoniobouortells.github.io/intro-macroeconomia/)

---

## 🚀 Estado del Proyecto

| Unidad | Título | Estado | Herramientas Clave |
|--------|--------|--------|-------------------|
| **Tema 1** | Mercado de Trabajo | ✅ Completo | WS-PS, Gini, Sindicatos, Tecnología |
| **Tema 2** | Bancos y Crédito | ✅ Completo | Valor Presente, Apalancamiento, Bonos |
| **Tema 3** | Fluctuaciones | ✅ Completo | Okun, Suavización, Multiplicador, Inversión |
| Tema 4 | Política Fiscal | ✅ Completo | Estabilizadores, Shocks, Conexión Laboral |
| **Tema 5** | Inflación | ✅ Parcial | Ecuación de Fisher, Phillips (Próx.) |
| **Tema 6** | Macro. Abierta | ⏳ Próximamente | Balanza de Pagos, Tipos de Cambio |

---

## 📁 Estructura del Repositorio

- `index.html`: Portal principal.
- `tema-1/`: Modelos de salarios, precios y desigualdad.
- `tema-2/`: Funcionamiento del sistema financiero y dinero.
- `tema-3/`: Ciclos económicos y determinantes de la demanda.
- `tema-4/`: El papel del gobierno y estabilización automática.
- `assets/`: Motores compartidos (Chart.js, KaTeX, CSS unificado).

---

## 🛠️ Tecnologías Utilizadas
- **HTML5/CSS3:** Diseño responsivo con estética Apple.
- **JavaScript (ES6+):** Lógica económica en tiempo real.
- **Chart.js:** Gráficos de alta fidelidad con latencia cero.
- **KaTeX:** Tipografía científica para todas las derivaciones matemáticas.
